Необходимо сделать конфигуратор MacBook по параметрам.
Референс: https://www.apple.com/ru/shop/buy-mac/macbook-pro/


Конфигуратор должен содержать в себе следующие параметры:
1.	Модель
2.	Цвет
3.	Размер (диагональ экрана)
4.	Процессор
5.	Оперативная память
6.	Графический процессор
7.	Накопитель
8.	Комплектация
9.	Результат:

Данные для конфигуратора: https://www.iport.ru/upload/front/data.json (признак комплекта - "bundle": true ; признак неактивного элемента - "add": false)
Меню, кнопки, форму заявки верстать и выводить не нужно, делаем только сам подбор.
Выбор каждого параметра необходимо реализовать на радиобаттонах. Выбор каждого параметра должен происходить последовательно. В результате нужно вывести блок с выбранными параметрами конфигурации, отдельно вывести комплектацию. 
Единственное ограничение по технологиям - использовать gulp (с обязательной возможностью собрать проект и запустить виртуальный веб-сервер). 
Готовый проект выложить на github.
