Это тестовое задание включает в себя:

1. Адаптивная верстка (320px - 1920px)
2. Запросы Rest API на PixabayAPI
3. Обработка ответа и вывод данных в шаблоне
4. Поиск на js

Используемые технологии:

1.  Pug
2.  Scss
3.  Webpack
4.  Gulp
5.  Vanilla JS
6.  Figma

Запуск проекта:

1. npm install
2. npm start

Скомплированная(build) версия

- в папке build
