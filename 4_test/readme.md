Это тестовое задание включает в себя:

1. Адаптивная верстка (320px - 1920px)
2. Авторизация по API
3. Кастомная валидация формы

Используемые технологии:

1.  Pug
2.  Scss
3.  Webpack
4.  Gulp
5.  Vanilla JS
6.  Photoshop
7.  Avocode

Запуск проекта:

1. npm install
2. npm start

- в папке build
